# Self Promise App

「自分との約束」を管理する、静かで優しいデザインのアプリケーション。

ユーザーにプレッシャーを与えず、日々の小さな約束を記録・達成することをサポートします。

## 特徴

- **軽い操作感**: 最小限のステップで約束を記録
- **優しいデザイン**: 評価や判断を求めない、自然なUI
- **シンプルな管理**: 約束の進捗を静かに追跡

## 技術スタック

- React + TypeScript
- TailwindCSS
- Vite

## セットアップ

```bash
npm install
npm run dev
```

## プロジェクト構成

```
src/
├── components/      # UI コンポーネント
├── pages/          # ページコンポーネント
├── hooks/          # カスタムフック
├── types/          # TypeScript型定義
├── utils/          # ユーティリティ関数
└── App.tsx         # メインアプリケーション
```
